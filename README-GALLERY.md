# 📸 Gupta Dental Care - Gallery Management System

## ✅ Setup Complete!

The gallery management system is now ready to use!

## 🚀 Quick Access

The server is running at: **http://localhost:3000**

- **Admin Panel (Upload Photos)**: http://localhost:3000/admin.html
- **Gallery Page**: http://localhost:3000/gallery.html  
- **Main Website**: http://localhost:3000/index.html

## 📝 How to Use

### Option 1: Upload via Admin Panel (Recommended)

1. Open http://localhost:3000/admin.html
2. Drag & drop images or click to browse
3. Click "Upload to Gallery"
4. Done! Images automatically appear on the gallery page

### Option 2: Manual Upload

1. Add images to the `Gallery` folder
2. Run: `npm run sync`
3. The gallery will update automatically

## 🎯 Features

✨ Drag and drop upload
✨ Multiple files at once
✨ Preview before upload
✨ Delete photos from admin
✨ Auto-updates gallery page
✨ Works with JPG, PNG, GIF, WebP

## 🛑 Stop Server

To stop the server, press `Ctrl + C` in the terminal

## 🔄 Restart Server

```powershell
npm start
```
